# キャッシュフロー計算書まとめ

## 全体像

- キャッシュフロー計算書は、一会計期間の現金及び現金同等物の増減を、原因別に示す財務諸表
- 資金の範囲
  - 現金
  - 現金同等物
    - 容易に換金でき、価値変動リスクが小さい短期投資
    - 例：取得日から満期日までの期間が3か月以内の定期預金
- 表示区分
  - 営業活動によるキャッシュフロー
  - 投資活動によるキャッシュフロー
  - 財務活動によるキャッシュフロー
- 同じ勘定科目でも、損益計算書や貸借対照表の分類ではなく、キャッシュの動きの原因で区分する

## 営業活動によるキャッシュフロー

- 営業損益計算の対象となる取引
- 投資活動・財務活動のどちらにも入らない取引
- 法人税等の支払額

### 間接法の基本

- 間接法では、税金等調整前当期純利益を出発点にして、現金の増減に直す
- 基本式
  - `税金等調整前当期純利益+減価償却費±営業活動に係る資産・負債の増減-法人税等の支払額`
- 減価償却費のような非資金費用は、費用として利益から引かれているが現金は出ていないので足し戻す

### 営業活動に係る資産・負債の増減

| 項目 | 増加した場合 | 減少した場合 | 考え方 |
| --- | --- | --- | --- |
| 売上債権 | マイナス | プラス | 売上は立ったが未回収なら現金は増えていない |
| 棚卸資産 | マイナス | プラス | 在庫が増えると仕入などで現金が出ている |
| 仕入債務 | プラス | マイナス | 未払いが増えると現金支出をまだしていない |

- 仕入債務は、商品や材料などを仕入れたが、まだ支払っていない代金
  - 代表例：買掛金

### 法人税等の支払額

- 損益計算書の法人税等は、発生した税金の費用
- キャッシュフロー計算書で引くのは、実際に支払った法人税等
- 未払法人税等がある場合
  - `法人税等の支払額=損益計算書の法人税等+前期末未払法人税等-当期末未払法人税等`
- 未払法人税等が増えている場合は、まだ支払っていない分が増えているため、実際の支払額は費用計上額より少なくなる

### 営業CFの計算例

損益計算書（P/L）

| 項目 | 額（単位：千円） |
| --- | ---: |
| 売上高 | 1,000,000 |
| 営業利益 | 200,000 |
| 法人税、住民税及び事業税 | 60,000 |
| 当期純利益 | 140,000 |

追加資料

| 項目 | 額（単位：千円） |
| --- | ---: |
| 減価償却費 | 30,000 |

貸借対照表（B/S）

| 項目 | 前期末（単位：千円） | 当期末（単位：千円） |
| --- | ---: | ---: |
| 売掛金 | 50,000 | 46,000 |
| 棚卸資産 | 30,000 | 33,000 |
| 買掛金 | 35,000 | 36,200 |
| 未払法人税等 | 30,000 | 30,000 |

この場合の営業活動によるキャッシュフローは、次のように計算する。

1. 税金等調整前当期純利益を見る

- この設例では営業外損益などがないため、営業利益と同額になる
- `200,000`

2. 減価償却費を足す

- 減価償却費は費用だが、現金は出ていないので足し戻す
- `200,000+30,000=230,000`

3. 売掛金の増減を調整する

- 前期末50,000、当期末46,000なので、4,000減少
- 売掛金が減るということは、売上代金を回収したということ
- 現金が増えるためプラス
- `230,000+4,000=234,000`

4. 棚卸資産の増減を調整する

- 前期末30,000、当期末33,000なので、3,000増加
- 棚卸資産が増えるということは、商品を仕入れて在庫が増えたということ
- 現金が出ているためマイナス
- `234,000-3,000=231,000`

5. 買掛金の増減を調整する

- 前期末35,000、当期末36,200なので、1,200増加
- 買掛金が増えるということは、まだ払っていない仕入が増えたということ
- 現金支出を抑えているためプラス
- `231,000+1,200=232,200`

6. 法人税等の支払額を引く

- 損益計算書では「法人税、住民税及び事業税」が60,000
- 貸借対照表では「未払法人税等」が前期末30,000、当期末30,000
- 未払いが増減していないため、実際に払った法人税等は当期計上額60,000
- `232,200-60,000=172,200`

したがって、営業活動によるキャッシュフローは `172,200` となる。

## 投資活動によるキャッシュフロー

- 長期的な資産運用や投資に関するキャッシュの動き
- 主な例
  - 固定資産の取得・売却
  - 現金同等物に含まれない有価証券や短期投資の取得・売却
  - 貸付け・貸付金の回収

### プラス・マイナスの整理

| 取引 | 投資活動によるCF |
| --- | --- |
| 有形固定資産を取得した | マイナス |
| 有形固定資産を売却した | プラス |
| 現金同等物に含まれない有価証券を取得した | マイナス |
| 現金同等物に含まれない有価証券を売却した | プラス |
| 貸付けを行った | マイナス |
| 貸付金を回収した | プラス |

### 有形固定資産売却益・売却損との関係

- 有形固定資産の売却による収入は、投資活動によるキャッシュフローに表示する
- 間接法の営業キャッシュフローでは、税金等調整前当期純利益に含まれている売却益・売却損を取り除く
  - 有形固定資産売却益：営業CFではマイナス調整
  - 有形固定資産売却損：営業CFではプラス調整
- 売却益・売却損は損益計算書上の損益であり、売却による実際の現金収入とは別
- マイナス調整する理由は、現金が減ったからではなく、投資活動の利益が営業CFに混ざっているため

例：

| 項目 | 金額 |
| --- | ---: |
| 有形固定資産の帳簿価額 | 1,000 |
| 売却額 | 1,600 |
| 売却益 | 600 |

- 営業CF
  - 税金等調整前当期純利益に含まれた売却益600を差し引く
- 投資CF
  - 実際の売却収入1,600をプラスする

### 現金同等物との区別

- 取得日から満期日までの期間が3か月以内の定期預金などは、現金同等物として資金の範囲に含まれる
  - 判定は「決算日から満期日まで」ではなく「取得日から満期日まで」
- 現金同等物に含まれるものは、キャッシュフロー計算書上の資金そのものなので、投資活動によるキャッシュフローの取得・売却にはしない
- 現金同等物に含まれない有価証券や短期投資の取得・売却は、投資活動によるキャッシュフローにする

## 財務活動によるキャッシュフロー

- 資金調達と返済に関するキャッシュの動き
- 主な例
  - 借入れ
  - 借入金の返済
  - 社債や株式による資金調達
  - 配当金の支払

| 取引 | 財務活動によるCF |
| --- | --- |
| 借入れを行った | プラス |
| 借入金を返済した | マイナス |
| 社債を発行した | プラス |
| 株式を発行した | プラス |
| 配当金を支払った | マイナス |

## 利息・配当金の表示

- 支払利息は、営業活動によるキャッシュフローに表示する方法と、財務活動によるキャッシュフローに表示する方法が認められている
- 表示方法の例
  - 受取利息・受取配当金・支払利息を営業活動、支払配当金を財務活動にする方法
  - 受取利息・受取配当金を投資活動、支払利息・支払配当金を財務活動にする方法

## 投資CFの計算例

次の取引があった場合、投資活動によるキャッシュフローはいくらか。

| 取引 | 金額 |
| --- | ---: |
| 有形固定資産の取得による支出 | 2,000 |
| 有形固定資産の売却による収入 | 1,300 |
| 貸付けによる支出 | 800 |
| 貸付金の回収による収入 | 500 |
| 取得日から満期日まで3か月以内の定期預金の取得 | 400 |

計算：

```text
-2,000+1,300-800+500= -1,000
```

- 取得日から満期日まで3か月以内の定期預金は現金同等物なので、投資活動によるキャッシュフローには含めない
- 投資活動によるキャッシュフローは `-1,000`
