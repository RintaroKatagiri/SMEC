# 法人税

* 法人税の企業活動で得た所得に対して課される税金
* 国税庁は法人税の所得金額を「益金の額ー損益の額」で計算するとされている
* キーワード
  * 欠損金
    * 税務上の赤字
    * 青色申告書を提出した事業年度の欠損金は一定のルールのもとで翌期以降に繰り越して損金算入できる
    * 原則として各事業年度開始の日前10年以内に関した事業年度の欠損金が対象
  * 青色申告
    * 税務署長の承認を受けると青色申告書で申告できる
  * 軽減税率
    * 中小企業に関しては、資本金などの基準をもとに軽減税率を定めるとしている

## 法人税の計算

- 法人税の計算にあたっての大前提として、「会社の利益≠税金計算の利益」
- 会計利益を税部上の利益に直す必要がある

### 税務調整

 法人税は、税引目当期純利益に税務調整（加算・減算調整）後の課税所得を計算し、その課税所得に法人税率を乗じて計算する

| 項目                   | 内容                                                                                                                       | 調整     |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------- | -------- |
| 受取配当金の益金不算入 | 法人が内国法人から配当等を受けた場合には、その受取配当等の額は、<br />課税所得の計算上益金に参入しない                     | 減算調整 |
| 交際費の損金不算入     | 法人が支出した交際費等の額のうち、一定の金額を超える部分の金額は、<br />課税所得の計算上損金の額に算入しない               | 加算調整 |
| 貸倒引当金の損金不算入 | 法人が設定した貸倒引当金の額のうち、一定の金額を超える部分の金額は、<br />課税所得の計算上損金の額に算入しない             | 加算調整 |
| 貸倒引当金の損金算入   | 損金不算入となった部分の金額のうち、債権の全額が回収不能など一定の条件を満たせば、<br />課税所得の計算上損金の額に算入する | 減算調整 |

### キーワード

- 不算入額
- 税部上の利益を計算するにあたって、足し引きする額
- 益金不算入額
  - 収益として計上しているが、税金では無視する　→　利益から引く
  - 受取配当金が該当する
- 損金不算入額
  - 費用として計上しているが、税金では認められない　→　利益に足す
  - 交際費、罰金、寄付金などが該当する
- 益金参入とか損金算入とかがあれば、それは素直に足したり、引いたりする（不算入だから処理が逆だった）
- これらをすべて計算した後に、法人税率をかけて法人税が算出される
