## キャッシュフロー

### キャッシュフロー計算書の全体像

- キャッシュフロー計算書は、一会計期間の現金及び現金同等物の増減を、原因別に示す財務諸表
- 資金の範囲
  - 現金
  - 現金同等物
    - 容易に換金でき、価値変動リスクが小さい短期投資
    - 例：取得日から満期日までの期間が3か月以内の定期預金
- 表示区分
  - 営業活動によるキャッシュフロー
    - 営業損益計算の対象となる取引
    - 投資活動・財務活動のどちらにも入らない取引
    - 法人税等の支払額
  - 投資活動によるキャッシュフロー
    - 固定資産の取得・売却
    - 現金同等物に含まれない有価証券や短期投資の取得・売却
    - 貸付け・貸付金の回収
  - 財務活動によるキャッシュフロー
    - 借入れ・借入金の返済
    - 社債や株式による資金調達
    - 配当金の支払
- 同じ勘定科目でも、損益計算書や貸借対照表の分類ではなく、キャッシュの動きの原因で区分する

### 営業キャッシュフロー

- 基本的な考え方
  - 間接法では、税金等調整前当期純利益を出発点にして、現金の増減に直す
  - `税金等調整前当期純利益+減価償却費±売上債権の増減±棚卸資産の増減±仕入債務増減-法人税等の支払額`
  - 減価償却費のような非資金費用は、費用として利益から引かれているが現金は出ていないので足し戻す
- 営業活動に係る資産・負債の増減
  - 売上債権：増加はマイナス、減少はプラス
    - 売上は立ったが未回収なら現金は増えていない
  - 棚卸資産：増加はマイナス、減少はプラス
    - 在庫が増えると仕入などで現金が出ている
  - 仕入債務：増加はプラス、減少はマイナス
    - 未払いが増えると現金支出をまだしていない
- 具体例
  - 損益計算書（P/L）
  - | 項目                     | 額（単位：千円） |
    | ------------------------ | ---------------- |
    | 売上高                   | 1,000,000        |
    | 営業利益                 | 200,000          |
    | 法人税、住民税及び事業税 | 60,000           |
    | 当期純利益               | 140,000          |
  - 貸借対照表（B/S）
  - | 項目         | 前期末（単位：千円） | 当期末（単位：千円） |
    | ------------ | -------------------- | -------------------- |
    | 売掛金       | 50,000               | 46,000               |
    | 棚卸資産     | 30,000               | 33,000               |
    | 買掛金       | 35,000               | 36,200               |
    | 未払法人税等 | 30,000               | 30,000               |
  - 計算
    1. 税金等調整前当期純利益（税引前当期純利益）を見る
    - この設例では営業外損益などがないため、営業利益と同額になる
    - 表から200,000
    2. 減価償却費を足す　
    - 減価償却費は費用だが現金は出ていないので足し戻す
    - 200,000+30,000=230,000
    3. 売掛金の増減
    - 前期末50,000、当期末46,000→4,000減少
    - 売掛金が減る＝売上代金を回収した→現金が増える→+4,000
    - 230,000＋4,000=234,000
    4. 棚卸資産の増減
    - 前期末30,000、当期末33,000→3,000増加
    - 棚卸資産が増える＝商品を仕入れて在庫が増えた→現金が出ていっている→-3,000
    - 234,000-3,000=231,000
    5. 買掛金の増減
    - 前期末35,000、当期末36,200→1,200増加
    - 買掛金が増える＝まだ払っていない仕入が増えた→現金支出を抑えている→+1,200
    - 231,000＋1,200=232,200
    6. 法人税等の支払額
    - 損益計算書では「法人税、住民税及び事業税」が60,000
    - 貸借対照表では「未払法人税等」に関して、前期末30,000、当期末30,000
    - 未払いが増減していない→実際に払った法人税等=当期計上額60,000→-60,000
    - 232,200-60,000=172,200

### 補足

- 有形固定資産の売却による収入は、投資活動のキャッシュフローとして扱われる
- 間接法で有形固定資産売却益が税金等調整前当期純利益に含まれている場合は、営業活動によるキャッシュフローでは差し引き、実際の売却収入は投資活動によるキャッシュフローに表示する
- 支払利息は、営業活動によるキャッシュフローに表示する方法と、財務活動によるキャッシュフローに表示する方法が認められている
