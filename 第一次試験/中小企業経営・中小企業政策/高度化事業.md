# 高度化事業

- 高度化事業は、工場団地・卸団地、ショッピングセンター、共同物流施設、商店街のアーケード・カラー舗装等の整備を行う中小企業組合等に対して、都道府県と中小企業基盤整備機構が協調して資金貸付と診断・助言を行う制度
- 基本的には中小企業単体ではなく、同じ目的を持つ企業同士で組織する中小企業組合等のグループを支援する
  - ただし、リニューアル事業などでは一社から高度化資金を利用できる場合がある
- 窓口は一般に都道府県であり、都道府県と中小機構が協調して貸付を行う

## 支援内容

### 貸付

- 長期・低利の固定利率による貸付
- 特別な法律の認定に基づく事業計画、災害復旧に係る貸付等では、無利子となる場合がある
- 貸付利率は毎年度見直される
  - 令和8年度（2026年度）貸付決定分は`1.35%`

### 診断・助言

- 貸付に際しては、事前に事業計画について専門的な立場から診断・助言を行う
- 事業計画の作成段階から、貸付後の経営アドバイスまで中小企業診断士等のサポートを受けられる

## 貸付条件

| 項目 | 内容 |
| --- | --- |
| 貸付対象者 | 中小企業組合等 |
| 貸付対象施設 | 土地、建物、構築物、設備など、資産計上されるもの |
| 貸付対象 | 貸付対象事業の実施に必要な施設整備資金。リニューアルを実施する場合を含む |
| 貸付期間 | 最長`20年以内`（うち据置期間`3年以内`） |
| 貸付割合 | 原則として貸付対象事業費の`80%以内` |
| 例外 | 関係法律の認定等を受けて実施する事業などでは`90%以内`の場合がある |

- `据置期間`とは、元本返済を猶予してもらえる期間のこと
  - 例：貸付期間20年、据置期間3年の場合、最初の3年間は元本返済を行わず、4年目以降に元本を分割返済するイメージ
  - 通常、据置期間中も利息の支払いは発生する

## 主な貸付対象事業

- `集団化事業`：工場や店舗などを集団で適地に移転する事業
- `集積区域整備事業`：商店街や工場等が集積している区域で、建物の建替え、道路舗装、アーケード等を整備する事業
- `共同施設事業`：共同加工施設、研究開発施設、物流センター、共同駐車場等を整備する事業
- `施設集約化事業`：店舗や工場を集約し、ショッピングセンターや共同工場などを整備する事業
- `グループ事業`：経営革新計画承認グループ、振興事業計画承認グループ等が共同利用施設等を整備する事業
- `リニューアル事業`：過去に高度化事業で整備した既存施設を再整備する事業

## 試験対策メモ

- 高度化事業は、補助金ではなく`貸付`
- 日本政策金融公庫ではなく、`都道府県と中小企業基盤整備機構`が協調して行う
- 原則は中小企業組合等のグループ支援であり、単体企業支援とは区別する
- 数字は、`20年以内`、`据置3年以内`、`80%以内`を優先して押さえる
- 診断・助言がセットである点も問われやすい
