# 参考問題6年度_年度差し替え対策

## 使い方

- 教材の6年度参考問題を、今年度版で差し替わりやすい論点に分解した索引。
- 問題文そのものではなく、`統計名`、`対象年`、`定義`、`大小関係`、`割合の桁感`を押さえる。
- 白書統計は出題年度によって対象年が変わるため、古い数値を固定暗記しない。

## 2026年版で確認した基礎値

出典は中小企業庁「2026年版中小企業白書」付属統計資料。

| 論点 | 押さえる値・関係 |
| --- | --- |
| 企業数 | 2021年、民営・非一次産業の中小企業は全体の`99.7%`、小規模企業は`84.5%` |
| 小規模企業の内訳 | 2021年、小規模企業のうち個人事業者は約`5割強` |
| 従業者数 | 2021年、中小企業は全体の`69.7%`。人数順は`中規模企業 > 大企業 > 小規模企業` |
| 付加価値額 | 2020年、中小企業は全体の`56.0%`、小規模企業は`14.5%` |
| 3業種の企業数 | 2021年、`小売業 > 建設業 > 製造業` |
| 小規模企業の付加価値額 | 2020年、製造業・小売業・卸売業では`製造業 > 小売業 > 卸売業` |
| 中小企業実態基本調査 | 2024年度決算実績では、製造業・卸売業・小売業の比較で、売上高経常利益率は`製造業`が最も高く、自己資本比率は`小売業`が最も低い |
| 開廃業率 | 統計によって定義が違う。雇用保険事業年報、経済センサス、会社設立登記を混同しない |

## 参考問題別の差し替え注意点

| 参考問題 | 主な論点 | 差し替え時に見るところ |
| --- | --- | --- |
| 第1問 | 企業規模別の従業者数 | 経済センサス。中小企業の従業者シェア、規模別の人数順 |
| 第2問 | 小規模企業の企業数・付加価値額 | 経済センサス。企業数は小規模企業が大半、付加価値額は企業数ほど高くない |
| 第3問 | 業種別の財務指標 | 中小企業実態基本調査。売上高経常利益率と自己資本比率の業種比較 |
| 第4問 | 設備投資額と投資目的 | 法人企業統計調査、法人企業景気予測調査。対象期間と選択肢の増減方向 |
| 第5問 | 業種別の所定内給与額 | 賃金構造基本統計調査。所定内給与と所定外給与を区別 |
| 第6問 | 資本金階層別の動向 | 法人企業統計。中小企業・中堅企業・大企業の資本金基準を確認 |
| 第7問 | 人材活用ガイドライン | 2023年6月改定、3つの人材戦略、3ステップ |
| 第8問 | エクイティ・ファイナンス | 返済不要、財務基盤安定、出資者支援、持株比率希薄化 |
| 第9問 | 海外展開・輸出 | 企業活動基本調査や民間調査。直接輸出、直接投資、労働生産性の比較 |
| 第10問 | 開業率・廃業率 | 雇用保険事業年報。長期推移と業種別の高低 |
| 第11問 | 企業年齢と雇用増加 | 経済センサス。若い企業ほど雇用増加に寄与しやすいかを確認 |
| 第12問 | 人材確保策 | 日本政策金融公庫調査など。賃上げ、シニア活用、長時間労働是正の回答割合 |
| 第13問 | 価格転嫁 | 取引条件改善状況調査。エネルギー、原材料、労務費で反映度が異なる |
| 第14問 | 業種別企業数 | 経済センサス。建設業・小売業・製造業の企業数順と増減 |
| 第15問 | 地域課題解決事業 | デジタル化・社会課題調査。取組有無、収支、資金調達方法 |
| 第16問 | 総資本営業利益率 | 法人企業統計。大企業・中堅企業・中小企業の資本金基準と推移 |
