# 小規模企業共済制度

- 小規模企業の経営者、役員、個人事業主等が、廃業や退職に備えて資金を準備するための共済制度
- いわば`経営者の退職金制度`
- 小規模企業共済法に基づく制度で、中小企業基盤整備機構が運営する
- 中小企業退職金共済制度や中小企業倒産防止共済制度（経営セーフティ共済）との引っかけに注意

## 加入できる者

- 個人事業主
- 個人事業主の共同経営者
  - 加入できる共同経営者は、個人事業主1人につき`2人まで`
- 小規模企業を経営している会社等の役員

### 従業員数の要件

| 業種等 | 常時使用する従業員数 |
| --- | :---: |
| 建設業、製造業、運輸業、不動産業、農業、宿泊業・娯楽業など | 20人以下 |
| 商業（卸売業・小売業）、宿泊業・娯楽業を除くサービス業 | 5人以下 |

- 主たる事業の業種で判断する
- 常時使用する従業員数は加入時点の人数要件であり、加入後に従業員数が増加しても共済契約は継続できる
- 中小企業退職金共済制度等の`被共済者`は加入できない

## 掛金

- 掛金月額は`1,000円〜70,000円`の範囲で、`500円単位`で設定できる
- 掛金は全額を`小規模企業共済等掛金控除`として、課税対象となる所得から控除できる
- 1年以内の前納掛金も同様に所得控除できる
- 掛金は共済契約者本人の収入から納付するため、事業上の損金または必要経費には算入できない

## 共済金の受取

- 共済金等の受取方法は次の3種類
  - `一括受取り`
  - `分割受取り`
  - `一括受取りと分割受取りの併用`
- 分割受取りや一括・分割併用を選ぶには、共済金の種類、年齢、金額などの要件を満たす必要がある

### 税法上の取扱い

| 受取・解約の形態 | 税法上の取扱い |
| --- | --- |
| 共済金（死亡を除く）または準共済金を一括で受け取る場合 | 退職所得 |
| 共済金を分割で受け取る場合 | 公的年金等の雑所得 |
| 一括・分割併用で受け取る場合 | 一括分は退職所得、分割分は公的年金等の雑所得 |
| 遺族が共済金を受け取る場合 | 相続税法上のみなし相続財産 |
| 65歳以上の任意解約等 | 退職所得 |
| 65歳未満の任意解約等 | 一時所得 |
| 12か月以上の掛金未払いによる機構解約 | 一時所得 |

- 解約手当金は、掛金納付月数が`12か月未満`の場合は受け取れない
- 任意解約では、掛金納付月数が`240か月（20年）未満`の場合、解約手当金が掛金合計額を下回る

## 契約者貸付制度

- 納付した掛金から算定した貸付限度額の範囲内で借入れできる
- `一般貸付`と、特別な事情がある場合の`特別貸付`がある
- 事業資金等を簡易・迅速に借りられる点が特徴

## 試験対策メモ

- 小規模企業共済制度：`経営者・役員・個人事業主等の退職金制度`
- 中小企業退職金共済制度：`従業員の退職金制度`
- 中小企業倒産防止共済制度：`取引先倒産による連鎖倒産防止`
- 小規模企業共済の掛金は`所得控除`であり、損金・必要経費ではない
- 共済金の受取は、一括なら原則`退職所得`、分割なら`公的年金等の雑所得`
