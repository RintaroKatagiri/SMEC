# 中小企業経営強化法

- 「経営革新計画」経営力向上計画」「事業継続力強化計画」「先端設備等導入計画」の根拠法規

## 経営革新計画

- 経営の向上を図るために新たな事業活動を行う経営革新計画の承認を受けることで、日本政策金融公庫の特別貸付制度や信用保証の特例など多様な支援を受けることが出来る

### 基本事項（問題の選択肢に関する内容であるため、経営革新計画とは関係ない内容もアリ）

- 先端設備等導入計画制度を規定した `生産性向上特別措置法`は令和3年度に廃止され、その内容は中小企業等経営強化法に統合された
- 地域未来投資促進法に主に規定されているのは、`地域経済牽引事業計画`
- `中小企業支援法`は、中小企業診断士の法的根拠

### 承認要件

| 事業期間終了時 | 「付加価値額」または<br />「従業員1人当たりの付加価値額」の伸び率<br />（年率3%以上） | 「給与支給総額」の<br />伸び率※<br />（年率1.5%以上） |
| -------------- | ------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| 3年計画の場合  | 9%以上                                                                                | 4.5%以上                                               |
| 4年計画の場合  | 12%以上                                                                               | 6%以上                                                 |
| 5年計画の場合  | 15%以上                                                                               | 7.5%以上                                               |

- 「給与支給総額」の伸び率は「付加価値額」または「従業員1人当たりの付加価値額」の伸び率の半分
    - `付加価値額＝営業利益＋人件費＋減価償却費`
    - `給与支給総額＝役員報酬＋給料＋賃金＋賞与＋各種手当`
        - 「各種手当」には、残業手当、休日手当、家族（扶養）手当、住宅手当等を含み、給与所得とされない手当（退職手当等）および福利厚生費は含まない

## 2026年時点の補足

### 経営革新計画

- 中小企業者等が`新事業活動`を行い、経営の相当程度の向上を図る計画
- 新事業活動の例
    - 新商品の開発または生産
    - 新役務の開発または提供
    - 商品の新たな生産方式または販売方式の導入
    - 役務の新たな提供方式の導入
- 申請先は、まず`都道府県`の商工担当部局に確認する
- 承認を受けても、融資・保証等の支援措置が自動的に受けられるわけではなく、各支援機関の審査が別途必要

### 経営力向上計画

- 人材育成、財務管理、設備投資など、自社の経営力を向上させるために実施する計画
- 認定を受けた事業者は、税制支援や金融支援等を受けることができる
- 経営力向上設備等を取得する計画では、申請前・設備取得前に`工業会等による証明書`または`経済産業大臣による確認書`を取得する必要がある

### 中小企業経営強化税制

- 適用期限は`2027年3月31日`まで
- 認定を受けた経営力向上計画に基づき対象設備を取得した場合、`即時償却`または`取得価額の10%の税額控除`を選択できる
    - 資本金3,000万円超の法人は税額控除`7%`
- 対象類型は、`A類型`、`B類型`、`D類型`、`E類型`
    - `C類型`は2025年4月1日をもって廃止

### 事業継続力強化計画

- 中小企業者等が策定する防災・減災の事前対策に関する計画
- 中小企業版の取り組みやすい`BCP`として位置づけられる
- 経済産業大臣の認定を受けると、税制措置、金融支援、補助金の加点などの支援を受けられる
- 複数事業者が連携して策定する`連携事業継続力強化計画`もある

### 先端設備等導入計画

- 中小企業者が労働生産性向上や賃上げ促進のために先端設備を導入する計画
- 市区町村が策定した`導入促進基本計画`に沿って、中小企業者が`先端設備等導入計画`を作成し、市区町村の認定を受ける
- 認定先端設備等導入計画に基づく設備投資は、市町村の判断により固定資産税の軽減措置を受けられる
    - 雇用者給与等支給額を`1.5%以上`増加させる方針を表明した場合：課税標準を`3年間1/2`に軽減
    - 雇用者給与等支給額を`3.0%以上`増加させる方針を表明した場合：課税標準を`5年間1/4`に軽減
    - 適用期限は`2027年3月31日`まで

### 認定経営革新等支援機関

- 税務、金融、企業財務に関する専門的知識や中小企業支援の実務経験を有する支援機関を国が認定する制度
- 経営力向上計画、先端設備等導入計画、補助金、税制等で、認定支援機関の確認や支援が関係することがある
