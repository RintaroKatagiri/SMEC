# 中小企業等経営強化法

- 中小企業者等の経営力向上、経営革新、防災・減災、先端設備導入などを支援する法律
- 試験では、計画ごとの目的、認定主体、数値要件、支援措置を区別する

## 主な計画

| 計画               | 主な目的                                                 | 認定・承認主体         |
| ------------------ | -------------------------------------------------------- | ---------------------- |
| 経営革新計画       | 新事業活動により経営の相当程度の向上を図る               | 都道府県等             |
| 経営力向上計画     | 人材育成、財務管理、設備投資などにより経営力を向上させる | 事業分野ごとの主務大臣 |
| 事業継続力強化計画 | 防災・減災の事前対策を進める                             | 経済産業大臣           |
| 先端設備等導入計画 | 労働生産性向上や賃上げ促進のため先端設備を導入する       | 市区町村               |

## 経営革新計画

- 中小企業者等が `新事業活動`を行い、経営の相当程度の向上を図る計画
- 新事業活動の例
  - 新商品の開発または生産
  - 新役務の開発または提供
  - 商品の新たな生産方式または販売方式の導入
  - 役務の新たな提供方式の導入
- 承認を受けると、日本政策金融公庫の特別貸付制度や信用保証の特例などの支援を受けられる場合がある
  - 承認だけで支援措置が自動的に適用されるわけではなく、各支援機関の審査が別途必要

### 承認要件

| 事業期間終了時 | 「付加価値額」または<br />「従業員1人当たりの付加価値額」<br />の伸び率 | 「給与支給総額」<br />の伸び率 |
| :--------------: | :---------------------------------------------------------------------: | :----------------------------: |
| 3年計画        |                                 9%以上                                 |            4.5%以上            |
| 4年計画        |                                 12%以上                                 |             6%以上             |
| 5年計画        |                                 15%以上                                 |            7.5%以上            |

- 年率では、付加価値額等が `3%以上`、給与支給総額が `1.5%以上`
- `付加価値額＝営業利益＋人件費＋減価償却費`
- `従業員1人当たりの付加価値額＝付加価値額÷従業員数`
- `給与支給総額＝役員報酬＋給料＋賃金＋賞与＋各種手当`
  - 残業手当、休日手当、家族手当、住宅手当などは含む
  - 退職手当、福利厚生費は含まない

## 経営力向上計画

- 人材育成、コスト管理、財務管理、設備投資など、自社の経営力を向上させるために実施する計画
- 事業分野別指針、または指針がない分野では基本方針を踏まえて策定する
- 認定を受けた事業者は、税制支援や金融支援等を受けられる場合がある
- 経営力向上設備等を取得する計画では、申請前・設備取得前に `工業会等による証明書`または `経済産業大臣による確認書`を取得する必要がある
- 申請先は事業分野ごとの主務大臣であり、中小企業庁では申請を受け付けない

## 中小企業経営強化税制

- 適用期限は `2027年3月31日`まで
- 認定を受けた経営力向上計画に基づき対象設備を取得・製作等した場合、`即時償却`または `取得価額の10%の税額控除`を選択できる
  - 資本金3,000万円超の法人は税額控除 `7%`
- 現行の対象類型
  - `A類型`：生産性向上設備
  - `B類型`：収益力強化設備
  - `D類型`：経営資源集約化設備
  - `E類型`：経営規模拡大設備
- 現行類型に `C類型`は含まれない
- `工業会等による証明書`または `経済産業大臣による確認書`は、設備取得前に申請する

## 事業継続力強化計画

- 中小企業者等が策定する、防災・減災の事前対策に関する計画
- 中小企業版の取り組みやすい `BCP`として位置づけられる
- 経済産業大臣の認定を受けると、税制措置、金融支援、補助金の加点などの支援を受けられる場合がある
- 複数事業者が連携して策定する `連携事業継続力強化計画`もある
- 事業継続力強化計画または連携事業継続力強化計画の認定を受けた事業者が、計画に記載された対象設備を取得等した場合、中小企業防災・減災投資促進税制の対象となる場合がある
  - 適用期限は `2027年3月31日`まで
  - 税制措置は `特別償却16%`

## 先端設備等導入計画

- 中小企業者が労働生産性向上や賃上げ促進のために先端設備を導入する計画
- 市区町村が策定した `導入促進基本計画`に沿って、中小企業者が計画を作成し、市区町村の認定を受ける
- 計画期間は `3年間〜5年間`
- 計画期間において、基準年度比で労働生産性が `年平均3%以上`向上することが必要
- 認定経営革新等支援機関による事前確認が必要
- 認定先端設備等導入計画に基づく設備投資は、市町村の判断により固定資産税の軽減措置を受けられる場合がある
  - 適用期限は `2027年3月31日`まで
  - 雇用者給与等支給額を `1.5%以上`増加させる方針を表明した場合：課税標準を `3年間1/2`に軽減
  - 雇用者給与等支給額を `3.0%以上`増加させる方針を表明した場合：課税標準を `5年間1/4`に軽減

## 認定経営革新等支援機関

- 税務、金融、企業財務に関する専門的知識や中小企業支援の実務経験を有する支援機関を国が認定する制度
- 先端設備等導入計画、補助金、税制等で、認定支援機関による確認や支援が関係することがある

## 試験対策メモ

- 経営革新計画：新事業活動、付加価値額等 `年率3%以上`、給与支給総額 `年率1.5%以上`
- 経営力向上計画：人材育成、財務管理、設備投資、主務大臣認定
- 事業継続力強化計画：防災・減災、経済産業大臣認定、中小企業版BCP
- 先端設備等導入計画：市区町村認定、労働生産性 `年平均3%以上`
- 税制の数字は、中小企業経営強化税制の `即時償却または10%税額控除`と、先端設備等導入計画の固定資産税 `3年間1/2`・`5年間1/4`を区別する
