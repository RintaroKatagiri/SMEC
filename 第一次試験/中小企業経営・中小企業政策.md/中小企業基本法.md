# 中小企業基本法　

## 中小企業者の範囲（定義）

| 業種分類                           | 定義（基準）                             |
| ---------------------------------- | ---------------------------------------- |
| 製造業その他（建設業、運輸業など） | 資本金3億円以下または従業員数300人以下   |
| 卸売業                             | 資本金1億円以下または従業員数100人以下   |
| 小売業、飲食店                     | 資本金5千万円以下または従業員数50人以下  |
| サービス業（宿泊業など）           | 資本金5千万円以下または従業員数100人以下 |

### 業界例から見た業種分類の目安

| 中小企業基本法上の分類 | 該当しやすい業界・事業例                                                                                                                                                         | 判断の目安                                                                     |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| 製造業その他           | 製造業、建設業、運輸業・郵便業（レッカー等を除く）、農林漁業、鉱業、金融・保険業、不動産業（駐車場等を除く）、電気・ガス・熱供給・水道業（電気小売・ガス小売を除く）、旅行業など | 卸売業・小売業・サービス業に明示されるもの以外。迷ったらまず「その他」と考える |
| 卸売業                 | 各種商品卸売、繊維・衣服卸売、飲食料品卸売、建築材料・鉱物・金属材料卸売、機械器具卸売など                                                                                       | 小売店・事業者・官公庁などに商品を大量または業務用として販売する               |
| 小売業                 | 各種商品小売、衣服小売、飲食料品小売、機械器具小売、無店舗小売、電気小売、ガス小売、飲食店、持ち帰り・配達飲食サービスなど                                                       | 消費者向けの販売。飲食店はサービス業ではなく小売業側で押さえる                 |
| サービス業             | 情報サービス、ソフトウェア、放送、映像・音声制作、広告制作、駐車場、物品賃貸、専門・技術サービス、宿泊、生活関連サービス、娯楽、教育、医療・福祉など                             | 役務提供が中心。宿泊業はサービス業、飲食店は小売業で区別する                   |

- 複数事業を営む場合は、`主たる事業`で業種分類を判断する
- 製造した商品をその場で販売するパン屋などの `製造小売`は、原則として `小売業`で押さえる
- 個別法・補助金では、ソフトウェア業、情報処理サービス業、旅館業などに別基準が置かれることがある

## 小規模事業者の範囲（定義）

| 業種分類                                   | 定義（基準）     |
| ------------------------------------------ | ---------------- |
| 製造業その他                               | 従業員数20名以下 |
| 商業（卸売業、小売業、飲食店）・サービス業 | 従業員数5名以下  |

- 資本金は基準に含まれないことがポイント

## 基本方針

1. 中小企業の経営の革新及び創業の促進並びに創造的な事業活動の促進を図ること
2. 中小企業の経営資源の確保の円滑化を図ること、中小企業に関する取引の適正化を図ること等により、中小企業の**経営基盤の強化**を図ること
3. 経済的社会的環境の変化に即応し、中小企業の経営の安定を図ること、事業の転換の円滑化を図ること等により、その変化への適応の円滑化を図ること
4. 中小企業に対する資金の供給の円滑化及び中小企業の**自己資本の充実**を図ること

## 条項

### 第2条2項（「経営の革新」の定義）

- この法律において「経営の革新」とは、新商品の開発又は生産、新役務の開発又は提供、商品の新たな生産又は販売の方式の導入、役務の新たな提供の方式の導入、新たな経営管理方法の導入その他の新たな事業活動を行うことにより、**その経営の相当程度の向上を図る**

### 第2条3項（「創造的な事業活動」の定義）

- この法律において「創造的な事業活動」とは、経営の革新又は創造の対象となる事業活動のうち、**著しい新規性を有する技術**又は**著しく創造的な経営管理方法**を活用したものをいう

### 第3条2項（基本理念）

- 中小企業の多様で活力ある成長発展に当たっては、小規模企業が、地域の特色を生かした事業活動を行い、就業の機会を提供するなどして**地域における経済の安定並びに地域住民の生活の向上及び交流の促進**に寄与するとともに、創造的な事業活動を行い、新たな産業を創出するなどして**将来における我が国の経済及び社会の発展**に寄与するという重要な意義を有するものであることに鑑み、独立した小規模企業者の自主的な努力が助長されることを旨としてこれらの事業活動に資する事業環境が整備されることにより、小規模企業の活力が最大限に発揮されなければならない

## 2026年時点の補足

- 中小企業基本法上の範囲は、中小企業政策における基本的な政策対象の `原則`
- 個別の法律・補助金・税制では、対象となる中小企業者の範囲が異なることがある
  - 例：みなし大企業の除外、業種別の特例、組合・NPO法人等の扱いなど
- 中小企業基本法に基づき、政府は毎年、`中小企業白書`を国会に提出する
  - 小規模企業については、小規模企業振興基本法に基づき `小規模企業白書`が作成される
