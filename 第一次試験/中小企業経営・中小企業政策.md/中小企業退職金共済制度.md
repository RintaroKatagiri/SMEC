# 中小企業退職金共済制度
- 独力では退職金制度を持つことが困難な中小企業について、退職金制度の整備を支援するもの
- 小規模企業救済制度や中小企業倒産防止救済制度（経営セーフティ共済）とは異なる
## 特徴
- 掛け金は全額非課税で、掛け金の負担軽減措置が設けられている
- 短期間労働者には、一般の従業員より低い特例掛金月額を設けている
- 初めて加入した事業主に対して掛金月額の2分の1（上限5000円）を従業員ごとに加入後4か月目から1年間国が助成する

## 2026年時点の補足

### 加入できる企業

- 常用従業員数または資本金・出資金のいずれかが一定範囲内であれば加入できる

| 業種 | 常用従業員数 | 資本金・出資金 |
| --- | ---: | ---: |
| 一般業種（製造業・建設業など） | 300人以下 | 3億円以下 |
| 卸売業 | 100人以下 | 1億円以下 |
| サービス業 | 100人以下 | 5,000万円以下 |
| 小売業 | 50人以下 | 5,000万円以下 |

- 個人企業や公益法人等は、常用従業員数で判断する
- 従業員は原則として全員加入
    - 期間雇用、試用期間中、休職期間中、短期間で退職することが明らかな者などは加入させなくてもよい

### 掛金

- 掛金は全額事業主負担
    - 従業員に一部でも負担させることはできない
- 掛金月額は`5,000円〜30,000円`の16種類から従業員ごとに選択できる
- 短時間労働者は、特例として`2,000円`、`3,000円`、`4,000円`の掛金月額も選択できる
- 掛金は法人では損金、個人企業では必要経費として全額非課税

### 国の助成

- 新規加入助成
    - 掛金月額の`2分の1`を、加入後4か月目から1年間助成
    - 従業員ごとの上限は`5,000円`
    - 短時間労働者の特例掛金月額には上乗せ助成がある
- 月額変更助成
    - 掛金月額18,000円以下の従業員の掛金を増額する場合、増額分の`3分の1`を1年間助成
    - 20,000円以上の掛金月額からの増額は助成対象外
