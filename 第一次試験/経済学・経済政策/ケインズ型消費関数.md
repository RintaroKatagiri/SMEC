# ケインズ型消費関数

ケインズ型消費関数は、家計の消費が主に現在の所得によって決まると考える消費関数である。

所得が増えれば消費も増えるが、増えた所得のすべてを消費に回すわけではない。増えた所得の一部は貯蓄に回るため、消費の増加幅は所得の増加幅より小さくなる。この考え方が、ケインズ型消費関数の中心である。

## 基本式

単純な形では、消費関数は次のように表される。

```text
C = C0 + cY
```

それぞれの意味は次のとおり。

| 記号 | 意味 |
| --- | --- |
| C | 消費 |
| C0 | 基礎消費、独立消費 |
| c | 限界消費性向 |
| Y | 所得 |

より厳密には、家計の消費は税金などを差し引いた後の可処分所得に左右されるため、`Y` の代わりに `Yd` を使って次のように書くこともある。

```text
C = C0 + cYd
```

試験問題では、税金を考えない単純なモデルなら `Y`、可処分所得を明示する問題なら `Yd` として読む。

## 基礎消費

基礎消費 `C0` は、所得がゼロでも必要になる消費である。

例えば、所得が一時的にゼロになっても、食費、住居費、光熱費など最低限の支出はすぐにはゼロにならない。このような所得に依存しない部分を基礎消費という。

グラフで見ると、基礎消費は縦軸の切片になる。

```text
消費 C
  |
  |        /
  |      /
  |    /
  |  /
  |/________________ 所得 Y
 C0
```

この切片が正であるため、所得が低いときには消費が所得を上回り、貯蓄がマイナスになることがある。これは、過去の貯蓄を取り崩したり、借入をしたりして生活する状態である。

## 限界消費性向

限界消費性向は、所得が1単位増えたときに、消費がどれだけ増えるかを表す。

```text
限界消費性向 = 消費の増加分 / 所得の増加分
MPC = ΔC / ΔY
```

基本式 `C = C0 + cY` では、`c` が限界消費性向である。

例えば、所得が100増えたときに消費が80増えるなら、限界消費性向は0.8である。

```text
MPC = 80 / 100 = 0.8
```

ケインズ型消費関数では、通常、限界消費性向は0より大きく1より小さいと考える。

```text
0 < c < 1
```

これは、所得が増えれば消費も増えるが、増えた所得をすべて消費するわけではない、という意味である。

## 平均消費性向

平均消費性向は、所得全体のうち、どれだけを消費に使っているかを表す。

```text
平均消費性向 = 消費 / 所得
APC = C / Y
```

消費関数 `C = C0 + cY` を使うと、平均消費性向は次のように整理できる。

```text
APC = C / Y
APC = (C0 + cY) / Y
APC = C0 / Y + c
```

ここで重要なのは、所得 `Y` が大きくなるほど `C0 / Y` が小さくなることである。つまり、ケインズ型消費関数では、所得が増えるほど平均消費性向は低下する。

一方、限界消費性向 `c` は、直線の傾きとして一定である。

## 限界消費性向と平均消費性向の違い

限界消費性向と平均消費性向は、似ているが見ているものが違う。

| 指標 | 見ているもの | 式 |
| --- | --- | --- |
| 限界消費性向 | 所得が増えた分のうち、どれだけ消費が増えるか | `ΔC / ΔY` |
| 平均消費性向 | 所得全体のうち、どれだけを消費しているか | `C / Y` |

基礎消費 `C0` が正であれば、平均消費性向は限界消費性向より大きくなる。

```text
APC = C0 / Y + c
```

`C0 / Y` が正なので、`APC > c` となる。

## 貯蓄関数

所得は、消費される部分と貯蓄される部分に分かれる。

```text
Y = C + S
```

したがって、貯蓄 `S` は次のように表せる。

```text
S = Y - C
```

消費関数 `C = C0 + cY` を代入すると、貯蓄関数は次のようになる。

```text
S = Y - (C0 + cY)
S = -C0 + (1 - c)Y
```

この式から、所得がゼロのときの貯蓄は `-C0` になることが分かる。基礎消費があるため、所得がゼロなら貯蓄はマイナスである。

## 限界貯蓄性向

限界貯蓄性向は、所得が1単位増えたときに、貯蓄がどれだけ増えるかを表す。

```text
限界貯蓄性向 = 貯蓄の増加分 / 所得の増加分
MPS = ΔS / ΔY
```

所得の増加分は、消費の増加分と貯蓄の増加分に分かれるため、次の関係が成り立つ。

```text
限界消費性向 + 限界貯蓄性向 = 1
MPC + MPS = 1
```

限界消費性向が0.8なら、限界貯蓄性向は0.2である。

```text
MPS = 1 - 0.8 = 0.2
```

## 損益分岐点の所得

消費と所得が等しくなる所得水準では、貯蓄がゼロになる。

```text
Y = C
S = 0
```

貯蓄関数 `S = -C0 + (1 - c)Y` で `S = 0` とおくと、貯蓄がゼロになる所得を求められる。

```text
0 = -C0 + (1 - c)Y
Y = C0 / (1 - c)
```

この所得より低いと、消費が所得を上回るため貯蓄はマイナスになる。この所得より高いと、所得が消費を上回るため貯蓄はプラスになる。

## 数値例

消費関数が次のように与えられているとする。

```text
C = 20 + 0.8Y
```

この場合、基礎消費は20、限界消費性向は0.8である。

所得が100なら、消費は次のようになる。

```text
C = 20 + 0.8 * 100
C = 100
```

所得が200なら、消費は次のようになる。

```text
C = 20 + 0.8 * 200
C = 180
```

所得が100から200へ増えると、所得は100増え、消費は80増えている。したがって、限界消費性向は0.8である。

平均消費性向は次のようになる。

| 所得 | 消費 | 平均消費性向 |
| ---: | ---: | ---: |
| 100 | 100 | 1.0 |
| 200 | 180 | 0.9 |

所得が増えると、消費額そのものは増える。しかし、所得に占める消費の割合である平均消費性向は低下している。

## 乗数との関係

ケインズ型消費関数は、乗数効果の理解にもつながる。

単純な経済で、総需要が消費 `C` と投資 `I` だけからなるとする。

```text
Y = C + I
```

ここに消費関数 `C = C0 + cY` を代入する。

```text
Y = C0 + cY + I
Y - cY = C0 + I
(1 - c)Y = C0 + I
Y = (C0 + I) / (1 - c)
```

このとき、投資が1増えると、均衡所得は `1 / (1 - c)` 倍だけ増える。

```text
投資乗数 = 1 / (1 - c)
```

限界消費性向が高いほど、乗数は大きくなる。なぜなら、最初に増えた所得の多くが消費に回り、その消費が別の人の所得になり、さらに次の消費につながるからである。

例えば、限界消費性向が0.8なら、投資乗数は5である。

```text
投資乗数 = 1 / (1 - 0.8)
投資乗数 = 5
```

限界消費性向が0.6なら、投資乗数は2.5である。

```text
投資乗数 = 1 / (1 - 0.6)
投資乗数 = 2.5
```

つまり、限界消費性向が大きい経済ほど、投資や政府支出の増加が所得を大きく押し上げる。

## グラフでの見方

消費関数のグラフでは、横軸に所得、縦軸に消費をとる。

- 縦軸の切片が基礎消費 `C0`
- 直線の傾きが限界消費性向 `c`
- 45度線と交わる点が、消費と所得が等しくなる点
- 45度線より上では、消費が所得を上回り、貯蓄はマイナス
- 45度線より下では、所得が消費を上回り、貯蓄はプラス

```text
消費 C
  |
  |              45度線 Y=C
  |            /
  |          /
  |        /  消費関数 C=C0+cY
  |      /  /
  |    /  /
  |  /  /
  |/__/________________ 所得 Y
 C0
```

試験では、式だけでなく、切片と傾きの意味をグラフで読めるようにしておくとよい。

## 試験での押さえ方

- ケインズ型消費関数は、消費が主に現在の所得に依存すると考える。
- 基本式は `C = C0 + cY` である。
- `C0` は基礎消費、`c` は限界消費性向である。
- 限界消費性向は、所得が1増えたときに消費がどれだけ増えるかを表す。
- 平均消費性向は、所得全体のうち消費に回る割合である。
- 基礎消費が正なら、所得が増えるほど平均消費性向は低下する。
- 貯蓄関数は `S = -C0 + (1 - c)Y` である。
- 限界消費性向と限界貯蓄性向の和は1である。
- 限界消費性向が大きいほど、乗数は大きくなる。

## よくあるひっかけ

- 「所得が増えると平均消費性向が上がる」とは限らない。ケインズ型の基本式では、所得が増えるほど平均消費性向は下がる。
- 「限界消費性向が1を超える」と考えない。通常は `0 < c < 1` である。
- 「消費額が増える」と「消費割合が増える」は別である。所得が増えれば消費額は増えるが、平均消費性向は低下することがある。
- `Y` が所得を表すのか、可処分所得を表すのかは、問題文の前提を確認する。
