# OECD加盟国との比較

OECD加盟国との比較では、OECD平均やOECD集計値が、各国の位置づけを読む基準になる。

ただし、OECD平均はそれ自体が1つの経済指標ではない。GDP、失業率、物価上昇率、租税負担率など、どの指標についての平均かを必ず確認する必要がある。

## OECDとは

OECDは、経済協力開発機構のこと。

2026年時点では38か国が加盟しており、日本、アメリカ、ドイツ、フランス、イギリス、韓国、オーストラリア、カナダなどが含まれる。

試験上は、OECDを「先進国・市場経済国との比較基準」として使うことが多い。ただし、OECD加盟国にはメキシコ、トルコ、コロンビア、コスタリカなども含まれるため、単純に「先進国だけの集まり」とは言い切れない。

## 比較基準の種類

OECDを基準に比較するときは、次の違いに注意する。

### 単純平均

各国の値を足して、国数で割る平均。

```text
OECD平均 = OECD加盟国の値の合計 / 加盟国数
```

各国を1国ずつ同じ重みで扱うため、小国も大国も同じ影響を持つ。

### OECD集計値

OECD加盟国全体を1つの地域のように合計して計算する値。

GDP、人口、失業者数などを合計してから比率を出すため、アメリカのような大国の影響が大きくなる。

```text
OECD集計値 = OECD加盟国全体を合計して計算した値
```

そのため、資料で「OECD average」「OECD total」「OECD」と書かれている場合、それが単純平均なのか、集計値なのかを確認する。

## 各国比較

### 1人当たりGDP

2024年の1人当たり名目GDP、購買力平価換算では、日本は主要国やOECD水準を下回る。

| 国・地域 | 1人当たりGDP、2024年 | 見方 |
| --- | ---: | --- |
| アメリカ | 86,147ドル | かなり高い |
| オーストラリア | 74,778ドル | 高い |
| ドイツ | 73,957ドル | 高い |
| カナダ | 65,483ドル | OECD単純平均に近い |
| OECD加盟国単純平均 | 65,149ドル | 38か国を同じ重みで平均 |
| イギリス | 62,824ドル | OECD集計値に近い |
| フランス | 62,524ドル | OECD集計値に近い |
| イタリア | 62,288ドル | OECD集計値に近い |
| OECD集計値 | 62,287ドル | 加盟国全体を合計した値 |
| 韓国 | 61,051ドル | OECD集計値をやや下回る |
| 日本 | 54,232ドル | OECD水準を下回る |

ポイントは、同じ「OECD」といっても、単純平均と集計値で値が異なること。2024年のこの指標では、加盟国単純平均は約65,149ドル、OECD集計値は約62,287ドルである。

### 失業率

2026年春時点の月次失業率では、日本と韓国はOECD水準より低く、フランスは高い。

| 国・地域 | 失業率 | 時点 | 見方 |
| --- | ---: | --- | --- |
| フランス | 7.7% | 2026年3月 | OECDより高い |
| OECD | 5.0% | 2026年3月 | 比較基準 |
| イギリス | 4.9% | 2026年1月 | OECDに近い |
| アメリカ | 4.3% | 2026年4月 | OECDより低い |
| ドイツ | 4.0% | 2026年3月 | OECDより低い |
| 日本 | 2.7% | 2026年3月 | 低い |
| 韓国 | 2.7% | 2026年3月 | 低い |

同じOECD比較でも、1人当たりGDPでは日本は低めに出る一方、失業率では日本は低い。したがって、「OECD平均より上か下か」は、指標ごとに意味が変わる。

## 試験での押さえ方

- OECD平均やOECD集計値は、OECD加盟国との国際比較に使う基準である。
- 「OECD平均を上回る」「OECD平均を下回る」という表現は、必ず指標名とセットで読む。
- 平均には、単純平均と集計値がある。
- 大国の影響を見たいときは集計値、各国を同じ重みで比べたいときは単純平均が使われやすい。
- 日本は、1人当たりGDPではOECD水準を下回るが、失業率ではOECD水準を下回る。
