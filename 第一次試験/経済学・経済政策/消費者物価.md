# 消費者物価

消費者物価とは、家計が購入する財・サービスの価格水準のこと。

物価の動きを見る代表的な指標が、消費者物価指数（CPI：Consumer Price Index）である。

## 消費者物価指数

消費者物価指数は、全国の世帯が購入する財・サービスの価格を総合し、物価の変動を時系列で測定する指数である。

日本では総務省統計局が毎月公表している。2026年5月時点で公表されている公式系列は、2020年を100とする2020年基準である。

```text
消費者物価指数 = 比較時点の購入費用 / 基準時点の購入費用 * 100
```

例えば、基準年に100で買えた同じ買い物かごが、現在110必要になれば、消費者物価指数は110になる。

## 計算の考え方

消費者物価指数は、基準年の消費構成を固定して、同じ買い物かごを買うために必要な金額がどれだけ変化したかを見る。

この考え方をラスパイレス方式という。

```text
物価上昇率 = (当期のCPI - 前期のCPI) / 前期のCPI * 100
```

前年同月比で見る場合は、当月のCPIを前年同月のCPIと比べる。

```text
前年同月比 = (当月CPI - 前年同月CPI) / 前年同月CPI * 100
```

## 主要な系列

### 総合指数

家計が購入する財・サービス全体の価格変動を見る指数。

最も広い範囲を対象にするため、生活実感に近い一方で、生鮮食品やエネルギーなど値動きの大きい品目の影響を受けやすい。

### 生鮮食品を除く総合

総合指数から生鮮食品を除いた指数。

日本で「コアCPI」と呼ばれることが多い。生鮮食品は天候などで価格が大きく変動しやすいため、基調的な物価を見るときに使われる。

### 生鮮食品及びエネルギーを除く総合

総合指数から、生鮮食品とエネルギーを除いた指数。

エネルギー価格は国際市況や為替の影響を受けやすいため、国内の基調的な物価動向をより見やすくするために使われる。

ただし、日本の「コアCPI」は「生鮮食品を除く総合」を指すことが多い。一方、海外では「食品とエネルギーを除く総合」をcore CPIと呼ぶことが多いため、何を除いているかを確認する。

## 対象になるもの

消費者物価指数は、家計が消費のために購入する財・サービスを対象にする。

代表的な費目は次のとおり。

- 食料
- 住居
- 光熱・水道
- 家具・家事用品
- 被服及び履物
- 保健医療
- 交通・通信
- 教育
- 教養娯楽
- 諸雑費

土地、株式、債券などの資産価格は、消費者物価指数の対象ではない。

## 他の物価指標との違い

### 企業物価指数

企業間で取引される財の価格を測る指数。

消費者が買う段階の価格ではなく、企業間取引の価格を見るため、原材料価格や輸入価格の変化が早く反映されやすい。

### GDPデフレーター

名目GDPと実質GDPの関係から求める物価指標。

国内で生産された財・サービス全体の価格変動を表すため、輸入品価格の扱いがCPIと異なる。

```text
GDPデフレーター = 名目GDP / 実質GDP * 100
```

### 消費者物価指数との違い

| 指標 | 主な対象 | 見るもの |
| --- | --- | --- |
| 消費者物価指数 | 家計が購入する財・サービス | 消費者が直面する物価 |
| 企業物価指数 | 企業間で取引される財 | 企業取引段階の物価 |
| GDPデフレーター | 国内で生産された財・サービス | GDP全体の価格変動 |

## 経済への影響

### インフレーション

物価が継続的に上昇すること。

需要が供給を上回って起こる需要インフレと、原材料費や賃金などの上昇で起こるコストプッシュ・インフレがある。

### デフレーション

物価が継続的に下落すること。

価格下落が続くと、企業収益、賃金、投資が伸びにくくなり、経済活動が停滞しやすい。

### 実質値への変換

名目値から物価変動の影響を除いたものが実質値である。

```text
実質値 = 名目値 / 物価指数 * 100
```

例えば、名目賃金が上がっていても、消費者物価がそれ以上に上がれば、実質賃金は下がる。

## 世界での比較

各国の消費者物価を比較するとき、消費者物価指数（CPI）の指数値をそのまま比べることはできない。

CPIは各国ごとに基準年を100として作る指数であり、日本のCPIが110、アメリカのCPIが120であっても、アメリカの物価水準が日本より高いとは限らない。

国際比較では、購買力平価（PPP）と為替レートから計算する比較物価水準を使う。

ここでは、OECDの家計最終消費支出の比較物価水準を使う。OECD=100なので、100を上回ればOECD平均より物価水準が高く、100を下回れば低い。

| 国・地域 | 消費者物価水準、2024年 | 見方 |
| --- | ---: | --- |
| アメリカ | 122.0 | OECDよりかなり高い |
| オーストラリア | 115.0 | OECDより高い |
| カナダ | 110.0 | OECDより高い |
| イギリス | 106.0 | OECDよりやや高い |
| OECD | 100.0 | 比較基準 |
| フランス | 94.6 | OECDよりやや低い |
| ドイツ | 92.6 | OECDよりやや低い |
| ユーロ圏 | 88.7 | OECDより低い |
| 日本 | 80.0 | OECDより低い |
| 韓国 | 78.0 | OECDより低い |
| トルコ | 47.2 | OECDよりかなり低い |

2024年の比較では、日本の消費者物価水準はOECDより低く、ユーロ圏よりも低い。一方、アメリカ、オーストラリア、カナダ、イギリスはOECDより高い。

注意点として、物価水準と物価上昇率は別の概念である。

例えば、トルコはインフレ率が高い国として知られるが、OECD基準で見た2024年の物価水準は低い。これは、物価が「どれだけ上がったか」と、国際的に見て「価格水準が高いか」は別だからである。

## 試験での押さえ方

- 消費者物価指数は、家計が購入する財・サービスの価格変動を示す。
- 日本のコアCPIは、通常「生鮮食品を除く総合」を指す。
- 生鮮食品及びエネルギーを除く総合は、より基調的な物価を見るために使われる。
- 資産価格は、消費者物価指数の対象ではない。
- CPIは消費者側の物価、企業物価指数は企業間取引の物価、GDPデフレーターはGDP全体の物価を表す。
- 名目値と実質値を区別するとき、消費者物価指数は物価変動を除くための代表的な指標になる。
- 国際的な物価水準を比べるときは、各国のCPI指数値ではなく、比較物価水準を見る。
