# 減税の乗数効果

## 基本

生産物市場の均衡条件が与えられている問題では、均衡所得 `Y` を税 `T` の式として整理し、税が変化したときに `Y` がどれだけ変化するかを見る。

もっとも基本的な形は次のとおり。

```text
Y = C + I + G
C = C0 + c(Y - T)
```

| 記号 | 意味 |
| --- | --- |
| Y | 国民所得、均衡所得 |
| C | 消費 |
| I | 投資 |
| G | 政府支出 |
| T | 租税 |
| C0 | 基礎消費、独立消費 |
| c | 限界消費性向 |
| Y - T | 可処分所得 |

税は所得から差し引かれるため、減税は可処分所得を増やし、消費を増やす。ただし、減税額のすべてが消費に回るわけではなく、最初に増える消費は `c × 減税額` である。

## 定額税の場合

税 `T` が所得に依存しない定額税である場合、均衡条件を整理すると次のようになる。

```text
Y = C0 + c(Y - T) + I + G
Y = C0 + cY - cT + I + G
(1 - c)Y = C0 - cT + I + G
Y = (C0 - cT + I + G) / (1 - c)
```

ここで、税 `T` が1増えると、均衡所得 `Y` は次の分だけ変化する。

```text
増税乗数 = -c / (1 - c)
```

減税は `T` の減少なので、減税額を正の値として表すなら、減税による所得増加は次のようになる。

```text
減税乗数 = c / (1 - c)
所得の増加額 = 減税額 × c / (1 - c)
```

例えば、限界消費性向 `c = 0.8` なら、減税乗数は4である。

```text
減税乗数 = 0.8 / (1 - 0.8)
減税乗数 = 4
```

10の減税を行うと、均衡所得は40増える。

```text
所得の増加額 = 10 × 4 = 40
```

## 政府支出乗数との違い

政府支出 `G` が1増えると、その1がそのまま総需要を増やす。

```text
政府支出乗数 = 1 / (1 - c)
```

一方、減税では、可処分所得が1増えても消費に回るのは `c` だけである。

```text
減税乗数 = c / (1 - c)
```

したがって、同じ限界消費性向なら、減税乗数は政府支出乗数より小さい。

```text
政府支出乗数 - 減税乗数
= 1 / (1 - c) - c / (1 - c)
= 1
```

## 減税乗数を大きくするもの

定額税モデルでは、減税乗数は次の式で決まる。

```text
減税乗数 = c / (1 - c)
```

したがって、減税の乗数効果を大きくするのは次の要素である。

- 限界消費性向 `c` が大きい
- 限界貯蓄性向 `1 - c` が小さい

理由は、減税によって増えた可処分所得のうち、消費に回る割合が大きいほど、次の所得増加と消費増加につながりやすいからである。

反対に、限界貯蓄性向が大きいと、減税で増えた所得が貯蓄に漏れるため、乗数効果は小さくなる。

## 比例税がある場合

問題によっては、税が所得に比例する形で与えられる。

```text
T = T0 + tY
```

この場合、`T0` は定額税部分、`t` は比例税率である。消費関数は次のようになる。

```text
C = C0 + c(Y - T)
C = C0 + c(Y - T0 - tY)
```

均衡条件を整理する。

```text
Y = C0 + c(Y - T0 - tY) + I + G
Y = C0 + c(1 - t)Y - cT0 + I + G
{1 - c(1 - t)}Y = C0 - cT0 + I + G
Y = (C0 - cT0 + I + G) / {1 - c(1 - t)}
```

定額税部分 `T0` を減税する場合の乗数は次のとおり。

```text
減税乗数 = c / {1 - c(1 - t)}
```

比例税率 `t` があると、所得が増えたときに税も増えるため、可処分所得の増加が抑えられる。これは自動安定化装置として働き、乗数を小さくする。

したがって、比例税を含むモデルで減税乗数を大きくするものは次のとおり。

- 限界消費性向 `c` が大きい
- 比例税率 `t` が小さい
- 限界貯蓄性向が小さい

## 輸入がある場合

輸入がある開放経済では、所得の増加の一部が輸入に向かうため、国内の需要として循環しにくくなる。

例えば、輸入が次のように表されるとする。

```text
M = M0 + mY
```

`m` は限界輸入性向である。均衡条件に輸入を含めると、乗数の分母に `m` が加わる。

```text
減税乗数 = c / {1 - c(1 - t) + m}
```

限界輸入性向 `m` が大きいほど、所得増加が海外への支出に漏れるため、国内所得への乗数効果は小さくなる。

したがって、輸入を含むモデルで減税乗数を大きくするものは次のとおり。

- 限界消費性向 `c` が大きい
- 比例税率 `t` が小さい
- 限界輸入性向 `m` が小さい
- 限界貯蓄性向が小さい

## 問題の解き方

生産物市場の均衡条件が与えられているときは、次の順序で考える。

1. `Y = C + I + G` などの均衡条件に、消費関数や税関数を代入する。
2. `Y` を含む項を左辺に集める。
3. `Y = ...` の形に整理する。
4. 税の項の係数を見る。
5. 減税額を正の値として見る場合は、符号を正に直して乗数を読む。

例として、次の式が与えられたとする。

```text
Y = C + I + G
C = 20 + 0.75(Y - T)
```

この場合、限界消費性向は `0.75` である。

```text
減税乗数 = 0.75 / (1 - 0.75)
減税乗数 = 3
```

20の減税を行うと、均衡所得は60増える。

```text
所得の増加額 = 20 × 3 = 60
```

## 選択肢の判断

「減税の乗数効果を大きくするもの」を選ぶ問題では、次のように判断する。

| 選択肢の内容 | 判断 |
| --- | --- |
| 限界消費性向が大きくなる | 減税乗数は大きくなる |
| 限界貯蓄性向が大きくなる | 減税乗数は小さくなる |
| 比例税率が大きくなる | 減税乗数は小さくなる |
| 限界輸入性向が大きくなる | 減税乗数は小さくなる |
| 基礎消費が大きくなる | 均衡所得の水準は上がるが、乗数そのものは変わらない |
| 投資が大きくなる | 均衡所得の水準は上がるが、乗数そのものは変わらない |
| 政府支出が大きくなる | 均衡所得の水準は上がるが、乗数そのものは変わらない |
| 減税額が大きくなる | 所得増加額は大きくなるが、乗数そのものは変わらない |

## よくあるひっかけ

- 減税乗数の符号に注意する。税 `T` の増加に対する乗数は負だが、減税額を正の値として見ると乗数は正になる。
- 減税乗数は、政府支出乗数より小さい。減税は可処分所得を増やすだけで、最初に需要へ回るのは `c` の部分だけだからである。
- 基礎消費、投資、政府支出の大きさは、均衡所得の水準を変えるが、線形モデルでは乗数の大きさを変えない。
- 比例税率や限界輸入性向は、所得増加が途中で漏れる割合を高めるため、乗数を小さくする。
- 問題文で `Y` が所得なのか、可処分所得なのかを確認する。`C = C0 + c(Y - T)` なら、税は消費を通じて効く。

## 試験での押さえ方

- 基本形は `減税乗数 = c / (1 - c)`。
- 限界消費性向が大きいほど、減税乗数は大きい。
- 限界貯蓄性向、比例税率、限界輸入性向が大きいほど、減税乗数は小さい。
- 「水準を変える要素」と「乗数を変える要素」を区別する。
